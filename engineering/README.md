# Engineering skills

Portable engineering-focused skill packages for Claude-style and Copilot-compatible installations.

| Skill | Purpose |
| --- | --- |
| `architecture-gcp-mvp-standards` | Opinionated MVP architecture guidance for TypeScript, Angular, GCP, Firebase, GitHub Actions, FinOps, and SRE visibility. |
| `angular-firebase-spa` | Angular SPA implementation guidance for standalone components, signals, Firebase Auth, guards, interceptors, and Firebase-backed frontends. |
| `firebase-functions-api` | Firebase Functions v2 TypeScript API guidance for callable/HTTP handlers, Admin SDK, custom claims, emulators, and backend patterns. |
| `firestore-security-rules` | Security-first Firestore and Storage rules authoring with emulator-backed tests. |
| `terraform-gcp-iac` | Terraform-first GCP infrastructure delivery guidance. |
